# Electrical-Circuit-Analysis-and-Simulation

## Overview

This repository contains Electrical Engineering projects focused on circuit analysis, theoretical validation, and MATLAB/Simulink-based simulation.

The projects demonstrate the application of fundamental electrical engineering concepts, analytical problem-solving techniques, and simulation-based verification of results.

---

## Projects Included

### 1. Superposition Theorem Circuit Analysis

#### Objective
To determine the current through a resistor in a multi-source DC circuit using the Superposition Theorem and verify the results using MATLAB Simulink.

#### Concepts Covered
- Superposition Theorem
- Circuit Simplification
- Ohm's Law
- Voltage Divider Rule
- MATLAB Simulink Verification

#### Key Result
- Current through 4Ω resistor = **2.5 A**

---

### 2. RL Circuit Power Analysis

#### Objective
To analyze the performance of a series RL circuit connected to a 200V, 50Hz AC supply and validate the theoretical calculations using MATLAB Simulink.

#### Concepts Covered
- AC Circuit Analysis
- Impedance Calculation
- Inductive Reactance
- RMS Values
- Active Power
- Reactive Power
- Apparent Power
- Power Factor
- MATLAB Simulink Validation

#### Key Results

| Parameter | Value |
|------------|---------|
| Supply Current | 20 A |
| Active Power | 2.4 kW |
| Reactive Power | 3.2 kVAR |
| Apparent Power | 4 kVA |
| Power Factor | 0.6 (Lagging) |

Simulation results closely matched analytical calculations with negligible error.
